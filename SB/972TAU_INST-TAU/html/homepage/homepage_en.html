What is DaTa?

DaTA is a search system that allows you to search all Tel Aviv University libraries collections, both printed and electronic. Please note: not all databases are accessible via this search system. For a more comprehensive search, we would advise searching directly via the databases available at Tel Aviv University.

Learn how to access full text of electronic resources off campus >>

Why should I Sign in?

Signing in to DaTA allows you: to view loan policy, check the list of active loans and requests, define interface language, and save your search(es) and interlibrary loan.
